## 🚀 Featured Project: アウフグース道場 (Aufguss Dojo)
熱波師（アウフギーサー）が自身の技術動画を投稿・アーカイブし、師範による承認やユーザー同士で評価（🔥いいね、難易度、段位など）を行う、技術特化型コミュニティプラットフォームです。

* **Architecture & Tech Stack**: Next.js (App Router), Supabase, Cloudflare (OpenNext / Wrangler), TailwindCSS, TypeScript
* **Key Features**:
  * **Product-Led Growth (PLG)**: システム自体がユーザーを連れてくるバイラルループの構築。
  * **マジック・インポート**: ユーザーがSNS（YouTube/X）のURLを貼るだけで、API経由で動画タイトルや説明文を自動補完し、投稿ハードルを極限まで下げる機能。
  * **動的OGP生成**: 投稿詳細がシェアされた際、技のステータス（難易度やタオルの種類）が記載された「道場認定トレーディングカード風」のOGP画像を動的に生成し、SNSでの承認欲求を満たす設計。
