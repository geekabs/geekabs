私は**サウナの熱波師（アウフギーサー）**として現場に立ちながら、AIエディタ（Cursor等）を駆使してWebサービスの**個人開発**を行っています。

元システムエンジニアとしての経験と、アウフギーサーとしての深いドメイン知識（現場のリアルな課題）を掛け合わせ、コードを直接書かずにAIエージェントに実装を委ねる次世代の開発スタイルで、本当に求められるプロダクトをハイスピードで創り上げています。

## 🚀 Featured Project: アウフグース道場 (Aufguss Dojo)
熱波師が自身の技術動画を投稿・アーカイブし、師範による承認やユーザー同士で評価（🔥いいね、難易度、段位など）を行う、技術特化型コミュニティプラットフォームです。自身の現場での課題解決から生まれた個人開発プロダクトです。

* **Architecture & Tech Stack**: Next.js (App Router), Supabase, Cloudflare (OpenNext / Wrangler), TailwindCSS, TypeScript
* **Key Features**:
  * **Product-Led Growth (PLG)**: システム自体がユーザーを連れてくるバイラルループの構築。
  * **マジック・インポート**: ユーザーがSNS（YouTube/X）のURLを貼るだけで、API経由で動画タイトルや説明文を自動補完し、投稿ハードルを極限まで下げる機能。
